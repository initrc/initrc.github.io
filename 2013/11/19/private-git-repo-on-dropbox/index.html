Private Git Repo on Dropbox

2013.11.19

Github is great, but there might be private things of which you want to keep a git history. If Github’s paid plan is an overkill, try git with Dropbox!

Keep the team size small. One is the best. It doesn’t sound good to have two commits pushed to Dropbox at the same time.
