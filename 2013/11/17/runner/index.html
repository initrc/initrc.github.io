Runner

2013.11.17

Intro

Runner is vim plugin that runs your code and outputs to the status bar.

I created Runner so that I don’t have to jump to another tmux pane to run some simple code, although it’s only 2.5 keys away to switch (Ctrl-a, j), 2 keys to run the code (up and enter, if ran before), and 2.5 keys to switch back to vim. And you have to move your hands to the right once. It’ll also be worse if the command to run the code varies.

Runner compiles/interprets single files. The supported languages are:

  • Java
  • C
  • C++
  • Python
  • Ruby

Now you can bind runner to one key and run them all! Perfect for code challenges.

Install

Vundle is the recommended plugin manager.

Add Bundle 'initrc/runner' to ~/.vimrc and run :BundleInstall.

Usage

Map the keyboard shortcut.

Changelog 0.1.0

  1. Add support for Java, C, C++, Python and Ruby
  2. <leader>r to run the code
