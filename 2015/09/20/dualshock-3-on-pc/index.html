DualShock 3 on PC

2015.9.20

I occasionally game on Windows on my MacBook, and I’ve been thinking about using DualShock 3 since I got one from the PS TV bundle. Previously I was using MotioninJoy and Better DS3 but it only worked when connected via USB. MotioninJoy does not support Apple’s bluetooth module. Found a better approach this weekend.

SCP DS3 is an open source driver that allows DualShock 2, 3 and 4 to work natively on Windows. I followed this post and it worked with USB easily. Then I used Zadig to override the driver so that any bluetooth module can work on SCP DS3.

SCP DS3 is much better than MotioninJoy/Better DS3. It runs as a service so you don’t need to run any app. It connects and works just like on PlayStation, and it’s easy to disconnect by long pressing L1 + R1 + PS. You can customize key bindings but the default one works fine as an Xbox 360 controller. It supports 4 controllers at the same time, and most importantly it’s open sourced. Never worry about MotioninJoy being a trojan anymore.
